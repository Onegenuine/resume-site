
# Vyacheslav — React Resume (Vite)

Это React-версия динамичного резюме, собранная на Vite + React + Framer Motion.

## Быстрый старт (локально)
1. Установка зависимостей: `npm install`
2. Запуск в режиме разработки: `npm run dev`
3. Сборка production: `npm run build`
4. Просмотр production локально: `npm run preview`

## Деплой
- Вercel: подключи репозиторий — Vercel автоматически развернёт проект (команда сборки `npm run build`).
- GitHub Pages: собери `build` (vite создаёт папку `dist`) и разверни её как статический сайт.

## Что включено
- React + Vite
- Framer Motion для анимаций
- Адаптивный современный UI (градиенты, glassmorphism)
- Страницы: Hero, About, Experience, Projects, Education, Contact
